## README

#### About

`tape.clj-template`

Tape Framework app template.

#### Usage

You must be familiar with [clj-new](https://github.com/seancorfield/clj-new) and have it installed.

```bash
> clj -X:new clj-new/create :template tape :name myname/myapp
```
